キチ/KICHI/    キツ/KITSU/    よし/yoshi/KICHI/キチ/    KITSU/キツ/    yoshi/よし/

good luck;  joy;  congratulations

Radical: 口 (mouth).   Strokes: 6画.   Elements: 口士.   Pinyin: jí.   Hangul: 길 [gil].   Nanori: え / き / きっ / きる / こし / と / よe / ki / ki' / kiru / koshi / to / yo.
